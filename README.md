#  Loja de Cookies e Café🍪☕ - Delicias Xamito

> Trabalho realizado pelos alunos do 3°P de Engenharia de Software sobre Sistema de Gestão de Pedidos e Integração Relacional.

---

## Sobre o Projeto

O trabalho consiste na criação de um sistema ficticio de cadastro e compras, e tambem emissões de relatorios gerais, aplicando os conhecimentos adquiridos ao longo do curso, principalmente os conceitos em JAVA e MySQL que são criterios de utilização do projeto.

---

## Linguagens Utilizadas

Aqui estão as principais ferramentas, linguagens e frameworks usados no desenvolvimento:

* JAVA
* MySql
* Não há a utilização de frameworks por questões de restrição do projeto.
---

## Funcionalidades

- Cadastro de clientes e produtos.
- Realização de compras.
- Emissão de relatorio.

---

## Como Executar o Projeto

### Pré-requisitos
Você vai precisar de:
* JDK (Java Development Kit) instalado.
* Uma IDE (como por exemplo o VScode).

> ### Passos para instalação:

1. **Baixe o projeto para o seu computador, ou realize o GitClone;**

2. **Configure e inicialize o Banco de Dados:**
   * Abra o programa que você usa para mexer no banco de dados;
   * Procure na pasta do projeto pelo arquivo de texto que contém os comandos do banco (Banco de Dados - script.sql);
   * Abra esse arquivo de texto, **copie todo o conteúdo** dele;
   * Cole esse código dentro do seu programa de banco de dados e clique em **Executar** (geralmente é um ícone de raio);

3. **Abra o projeto no seu programa de programação (IDE), e execute o sistema;**

---

### Atenção ao usar o sistema (Preenchimento de Valores)

Ao cadastrar ou digitar preços e valores no sistema, atenção à formatação:
* **Use sempre VÍRGULA ( , )** para separar os centavos. 
* *Exemplo correto:* 15,90 ou 99,00
* **Não utilize ponto ( . )**, caso contrário o sistema pode apresentar erro.
